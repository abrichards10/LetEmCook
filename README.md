Instructions on how to install and set up your Flutter project.

```sh
# Clone the repository
git clone https://github.com/abrichards10/LetEmCook.git

# Navigate to the project directory
cd let_em_cook

# Get dependencies
flutter pub get

# Run the project
flutter run
```
